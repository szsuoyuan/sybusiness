  • 授权设置
  • 菜单设置

1. 要在微信公众平台“开发模式”下使用自定义菜单，首先要在公众平台申请自定义菜单使用的AppId 和AppSecret，然后填入下边表单。

2. 提交完id和密钥后，可以在【菜单设置】中设置各个菜单项，然后进行发布，您的微信公众号便支持 自定义菜单了。

3. 公众平台规定，菜单发布24小时后生效。如果为新增粉丝，则可马上看到菜单。

应用ID：
* 公众平台申请到App ID
应用密匙：
* 公众平台申请到App Secret
*

1.主菜单至少开启1个，最多只能开启3个；2级子菜单最多开启5个，可不开启

2.如果尚未在授权设置中填写在公众平台申请自定义菜单使用的“AppId”和“AppSecret”，只能保存自定义菜单设置，不能生成自定义菜单！

注：自定义菜单使用的“AppId”和“AppSecret”在微信公众平台-功能-高级功能-开发模式页面获取

3.生成自定义菜单,必须在已经保存的基础上进行,临时添加或勾选启用后点击生成是无效的! 第一步必须先修改保存状态！第二步点击生成!

4.当您为自定义菜单填写链接地址时必须以"http://"开头

5.当您为自定义菜单填写关键字时只能填写一个关键字

6.添加主菜单时，默认状态为不启用状态；添加子菜单时，默认状态为启用状态，生成菜单时请注意菜单启用状态的勾选

7.添加菜单时显示顺序值默认都为0，当都为0时按添加时间排序，新添加的排在最后；当更改数字时（>=0）,按数字大小排序，数字越大越往后排
